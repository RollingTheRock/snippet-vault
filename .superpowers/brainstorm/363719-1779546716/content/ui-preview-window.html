预览窗口技术细节

渲染方式
  • 每个预览窗口是一个独立 BrowserWindow
  • 通过 data URL 或 blob URL 注入代码
  • HTML 片段自动包裹完整文档结构
  • CSS 和 JS 直接内联注入
安全与限制
  • 禁用外部网络请求（CSP 限制）
  • 禁用 alert/confirm/prompt
  • 窗口间代码隔离，互不干扰
  • console.log 输出捕获到主窗口
