窗口风格说明

  • 标题栏：采用应用内自定义标题栏，不模拟 macOS 或 Windows 原生样式，保持跨平台一致性
  • 关闭按钮：右上角 "x" 按钮，点击关闭当前预览窗口
  • 图标：使用简洁的符号/文字按钮替代 emoji，保持专业感
  • 框架：Electron BrowserWindow 的 frame 设为 false，完全自定义标题栏，适配 Fedora / GNOME / KDE
